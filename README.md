# O Que Devo Jogar Hoje

Projeto desenvolvido durante as aulas de Programação II 2025.2.  
Seu objetivo é ser um aplicativo web de informação e recomendação de jogos.

## Como Instalar

- _É necessário ter NodeJS instalado em seu computador_
- Clone o repositório para uma pasta de seu computador.
- Estando neste diretório, execute 'npm install' no prompt de comando.
- Após concluir a instalção das dependências, execute o comando 'npx nodemon server.js'.
