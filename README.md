# O Que Devo Jogar Hoje

Projeto desenvolvido durante as aulas de Programação II 2025.2.  
Seu objetivo é ser um aplicativo web de recomendação de jogos.

## Como Instalar

- _É necessário ter NodeJS instalado em seu computador_
- Clone o repositório para uma pasta de seu computador.
- Estando neste diretório, em um editor de código, execute os seguintes comandos em um terminal:

1. cd FrontEnd
2. npm install
3. npm run dev

- Mantendo este terminal aberto, abra um novo e execute os próximos comandos:

1. cd BackEnd
2. npm install
3. npx nodemon server.js

- Após isso acesse http://localhost:5173/ e pronto!

- Foram utilizadas as bilicotecas:
  - @mui/material
  - @mui/icons-material
  - axios
  - cors
  - express
  - express-session
  - passport
